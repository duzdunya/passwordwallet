## Installation
```
git clone https://github.com/duzdunya/passwordwallet.git

cd passwordwallet

python3 -m venv venv

# Windows
venv/Scripts/activate

# Linux
source venv/bin/activate

# Install requirements
pip install -r requirements.txt

# start the application
python3 main.py
```
